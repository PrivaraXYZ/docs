# Privara

**The Consumer App for Programmable Stablecoins**

Built on ReineiraOS. Web3 infrastructure. Web2 experience.

---

## Overview

Privara is the consumer application layer for [ReineiraOS](./lightpaper.md) — programmable infrastructure for stablecoins built on Arbitrum and powered by Fhenix FHE.

Privara delivers a familiar payments experience while the protocol handles confidentiality, escrow, and cross-chain operations underneath.
